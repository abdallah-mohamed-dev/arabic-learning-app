# تطبيق تعلم اللغة العربية

مشروع ويب لتعلم اللغة العربية مبني بـ Next.js و Tailwind CSS.

## الوصف

- التطبيق يعرض دروسًا في اللغة العربية من خلال صفحة الدروس.
- كل درس يحتوي على صور وعناصر تعليمية تساعد المتعلم على الفهم.
- يوجد اختبار لكل درس لقياس الفهم والحصول على نتيجة.
- النتائج تظهر في صفحة منفصلة بعد إكمال الاختبار.

## بنية المشروع

- `app/` - يحتوي على صفحات التطبيق وواجهته.
- `components/` - يحتوي على مكونات إعادة الاستخدام مثل بطاقات الدرس والأسئلة.
- `data/lessons.json` - يحتوي على بيانات الدروس المستخدمة في التطبيق.
- `public/images/` - يحتوي على صور الدروس.

## كيفية التشغيل

1. تثبيت الحزم:
   ```bash
   npm install
   ```
2. تشغيل التطبيق محليًا:
   ```bash
   npm run dev
   ```
3. فتح المتصفح على:
   ```
   http://localhost:3000
   ```
